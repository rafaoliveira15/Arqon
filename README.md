# SPRINT 1

A Sprint 1 corresponde à fase inicial de planejamento do projeto Arqon.
Nesta etapa foram definidos os requisitos do sistema, identidade visual e estrutura inicial da plataforma.

O sistema tem como objetivo desenvolver uma plataforma digital para locação de roupas premium sustentáveis, permitindo que usuários acessem peças de marcas renomadas de forma prática e moderna.

O projeto contempla cinco páginas principais:

* Página de Login
* Cadastro de Marcas Parceiras
* Cadastro de Peças
* Painel Administrativo
* Listagem de Produtos

O objetivo é desenvolver um sistema acessível, organizado e eficiente para gerenciamento de peças de vestuário e marcas parceiras.

---

## Funcionalidades

* Login seguro com e-mail e senha
* Cadastro de marcas parceiras
* Cadastro de produtos (peças de roupa)
* Registro de informações detalhadas das peças
* Listagem organizada dos produtos cadastrados

---

## Escopo

Diante das demandas definidas para a Sprint 1, a equipe Arqon assume o desenvolvimento de cinco telas essenciais para o sistema:

Página de Login
Campos para e-mail e senha com botão de autenticação.

Painel Administrativo
Área para gerenciamento de marcas e produtos cadastrados.

Cadastro de Marca Parceira
Campos para nome da marca, e-mail de contato e telefone.

Cadastro de Peça
Campos para:

* marca parceira
* título da peça
* descrição do produto
* valor da locação diária

Listagem de Peças
Interface para visualização organizada dos itens cadastrados.

---

## Considerações

### Escalabilidade

A estrutura foi planejada para permitir expansões futuras como:

* histórico de pedidos
* relatórios administrativos
* melhorias na experiência do usuário

### Segurança

As senhas dos administradores devem ser armazenadas utilizando criptografia para garantir a segurança das informações.
